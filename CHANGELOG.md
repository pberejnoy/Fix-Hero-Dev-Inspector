# FixHero Dev Inspector Changelog

## Version 0.2.0 (2023-04-07)

### 🚀 Major Enhancements

#### UX/UI Improvements
- **Quick Status Panel**: Added live session state indicator and issue counter
- **Pinned Tools**: Added customizable favorites section for quick access to most-used tools
- **Sticky Export Panel**: Added persistent export options at the bottom of the screen
- **AI Summary Badge**: Added visual indicator when AI suggestions are available for an issue
- **Share Dialog with QR Code**: Added ability to share sessions via link or QR code
- **Keyboard Shortcuts Customizer**: Added ability to configure custom keyboard shortcuts

#### Multi-Role Support
- **Role Selector**: Added ability to switch between Frontend, Backend, Designer, and All Roles views
- **Role-specific Tools**: Implemented tools tailored to different user roles
- **Contextual UI**: UI adapts based on selected role

#### Frontend Developer Tools
- **Responsive Tester**: Test websites at different screen sizes
- **Contrast Checker**: Verify color contrast for accessibility compliance
- **Grid Visualizer**: Visualize and generate CSS grid layouts
- **CSS Inspector**: Inspect and edit CSS properties
- **Typography Checker**: Analyze and test typography settings
- **Framework Helpers**: Reference for CSS frameworks

### 🔧 Other Improvements
- **Dark Mode**: Added support for dark mode
- **Performance Optimizations**: Improved loading times and reduced memory usage
- **Enhanced Error Handling**: Better error messages and recovery
- **Improved Accessibility**: Better keyboard navigation and screen reader support

### 🐛 Bug Fixes
- Fixed issue with screenshot capture on high-DPI displays
- Fixed memory leak in the element inspector
- Fixed inconsistent sync status indicators
- Improved error handling for network requests
