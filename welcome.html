Welcome to FixHero Dev Inspector

Thank you for installing FixHero Dev Inspector! This powerful tool helps you inspect, debug, and document web applications with ease.

Key Features

  • Element Inspection: Easily inspect and capture details about any element on a webpage.
  • Screenshot Capture: Take screenshots of the entire page or specific elements.
  • Note Taking: Add notes about issues or features you discover.
  • Issue Tracking: Keep track of bugs and issues you find during development.
  • Firebase Integration: Sync your data across devices (optional).

Keyboard Shortcuts

Take Screenshot

Ctrl + Shift + S (Windows/Linux) or Command + Shift + S (Mac)

Add Note

Ctrl + Shift + N (Windows/Linux) or Command + Shift + N (Mac)

Toggle Sidebar

Ctrl + Shift + D (Windows/Linux) or Command + Shift + D (Mac)

Open Dashboard

Ctrl + Shift + M (Windows/Linux) or Command + Shift + M (Mac)

Click the FixHero Dev Inspector icon in your browser toolbar to get started!
